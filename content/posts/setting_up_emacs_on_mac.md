## Dired {#dired}

Setting up Dired requires the GNU version of `ls`. One can install it via Homebrew:

```bash
brew install coreutils
```


## OpenPGP {#openpgp}

Download GnuPG using Homebrew:

```bash
brew install gnupg
```
